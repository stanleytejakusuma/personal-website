Into the boba we go!

$BOBA is a compute-focused **Optimistic Rollup** @optimismPBC. Boba uses the Turing hybrid compute to achieve max capabilities of ETH ecosystem.

\*Technical Details

Turing performs using L2TGeth – Go Ethereum. Enabling devs to use single line function calls.

<img alt="" src="/musings/boba-network-thread/image-01.webp" width="1024" height="576" loading="lazy" decoding="async" />

(source: [<u>here</u>](https://boba.network/turing-hybrid-compute/))

Additionally, Boba added functionalities atop of Optimism:

- providing new cross-chain messaging

- different gas price logic, rapid L2 -\> L1 exits (without 7d delay)

- public fraud detector interacting L2 ETH using normal ETH methods

- organized as a DAO

- native NFT bridge

Thanks to the augmented Optimistic Rollup, gas price on Boba L2 change every 30s with 5% max percent change (i.e. gasPrice is 10 Gwei → the next price will be around 9.5 - 10.5 Gwei)

Slightly deviating from Boba, but Optimistic upgrade to v2 introduces several changes:

- Contract source code must be verified

- Verified source code will be recompiled with standard Solidity compiler

- ETH no longer ERC-20 compatible

- Altered Boba fee scheme

- EOAs no longer contract wallets

- Gas usage decrease

For those interested in developing an app in Boba, use **The Graph**! It’s an indexing protocol for organizing blockchain data with a service – GraphQL. Just connect to an API through Boba and Voilà!

Supporting the network, $BOBA is the native token with the main use cases as **governance** and **staking**.

$BOBA has a fixed total supply of 500M, currently ~150M in circulation. Remaining amounts are linearly released over the next 4 years. Those that receive BOBA in TGE are $OMG hodlers.
