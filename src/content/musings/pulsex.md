- PulseX is the **main DEX** for PulseChain.

- Brief information about PulseChain:

1.  Built to solve Ethereum’s recent update which increases transaction cost in the long term.

2.  The drawback of Ethereum’s high transaction cost would hugely impact the token HEX – the first token that has a Blockchain Certificate of Deposit and currently the highest interest amongst other blockchain.

3.  TL;DR a better version of Ethereum.

- PulseX, PulseChain, and HEX is created by **Richard Heart**:

1.  Good track record, he created HEX (world’s first blockchain Certificate of Deposit). Price action of HEX – price has gone more than 10,000x.

2.  PulseChain is a copy of ETH but faster and more efficient. Largest Airdrop.

- To buy PulseX’s native token (PLSX), users must ‘sacrifice’ their eligible tokens.

- Sacrificing is a new method of purchasing tokens before the token is available in the open market:

1.  Users could only buy the token during a specific sacrifice time frame

2.  The users would receive sacrifice points, which are 1:1 to the target token

3.  The first 5 days have the best rate, afterwards the sacrifice cost would increase

4.  For example, for $1 users receive 10,000 SPs. However, users would require $9.9 to receive that same amount on the last day before sacrifice phase ends

5.  At the end of the period, users could receive a bonus for the amount of USD sacrificed. The more USD value a user sacrifices, the higher the bonus.

6.  Transacting sacrifice is recommended on non-custodial wallets as the target token would be sent there.
