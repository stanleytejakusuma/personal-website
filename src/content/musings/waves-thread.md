The notion behind the project is that the development of smart contracts should be **as easy as** launching a conventional web app!

Now, why WAVES?

- Easy to issue tokens

- Easy to develop smart contracts

- Leased PoS

- Performance from Waves-NG

- Open source code

- An ecosystem for developers!

Diving deeper into the network, WAVES employs nodes to secure the protocol. These nodes must have a minimum balance of **1000 WAVES** ($50k USD).

WAVES is an account-based model protocol that has a functionality for transaction or managing smart contracts.

dApps here is the account with the dApp script assigned. This account has permission to add, modify, or delete; transfer tokens; issue, reissue, burn, and sponsorship setup.

WAVES supports both smart contract tokens and NFT.

Issuance of smart contract tokens requires the script, minimum of 1 WAVES for tx fee, and increased by 0.004 WAVES for each smart asset involved.

Whereas, the issuance for NFT has a minimum fee of 0.001 WAVES.

Now, into the network architecture! WAVES splits the network into three parts: Mainnet, Testnet, and Stagenet.

Stagenet is the experimental network where WAVES team test protocol functionalities before deployment. It is generally more unstable and blockchain height rollbacks are much more likely.

The WAVES chain itself uses:

- *Blake2b256* and *Keccak256* to create crypto hashes

- *Curve25519* to create and verify signatures

- *Base58* to create strings/information form of bytes

Another implementation WAVES leverages on is the Waves-NG, which splits the block into key blocks and micro blocks.

The process is as follows: miner node create block → sends the key block (no tx) → sends micro block (contain tx, similar to normal block) with mining time interval of 3s → miners will mine those micro blocks and propagate them to the network.

To the asset that powers the network itself – $WAVES!

The token began as a fixed-cap token with 100M max supply. All the tokens are in circulation. $WAVES has a main use case as the native token for the protocol.

Around 20M tokens are issued as the block rewards. The initial block reward has been set to 6 WAVES, every 100k blocks miners could decrease or increase the block reward by 0.5 WAVES.

Currently, the APR of block rewards are 3.61% for lease waves and 3.68% for mine waves.
